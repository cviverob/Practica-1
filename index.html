Elmo cines

Nuestra página web va a tener una gestióm de películas, para poder darlas de alta y baja, podiendo incluir una foto, trailer, título, sinopsis, duración, étc. Por otro lado, estará la gestión de la

cartelera, para poder modificar los horarios de las películas de cada cine de la empresa y coordinar las sesiones de las salas. Adicionalmente tendrá la gestión de las salas, para poder darlas de alta

y de baja, con el tamaño y asientos que queramos. Todo esto estará a cargo del staff, mientras que un usuario normal podrá consultar las películas y comprar las entradas para una sesión específica.

Los usuarios registrados en la web tendrán acceso a un perfil personal para poder consultar sus puntos de socio, historial de compra, étc.

Aquí puedes acceder a los bocetos, contacto, detalles, miembros y planificación.
