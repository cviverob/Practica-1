Menu principal

Este es la página principal, la cual ve el usuario nada más entrar. Contiene el logo de la empresa, un botón para registrarse, las películas en emisión, y un pie de página con datos de contacto y miembros.

  1. Este botón te redirige a la pantalla de conexión y registro.
  2. Al pulsar encima de la imagen te lleva a la información de la sesión.
  3. En el pie de página puedes comunicarte con los desarrolladores con el botón de contacto.

Consulta de sesiones

Este boceto muestra toda la información de la película y sus sesiones

  1. Estos botones indican las fechas disponibles para la sesión
  2. Información sobre la película (nombre, imagen, sinopsis, etcétera).
  3. Botones para seleccionar la hora de la sesión.
  4. Al pulsar el botón de continuar nos permite ir a la pantalla de selección de butacas.

Selección de butacas

En esta pantalla el usuario podrá seleccionar sus asientos siempre que estén disponibles. Para avanzar de este paso el usuario deberá estar logeado, sino, se le mostrará un mensaje solicitando su registro.

  1. Al pulsar en los asientos de seleccionarán los asientos.
  2. Si el usuario está registrado, se procederá con el proceso de compra, llevándole al menú principal y generando un pdf con su entrada.

Perfil de usuario

Esta sección muestra los bocetos del logeo, del registro, y del historial de compra del usuario.

Conexión y registro

Este boceto muestra la pantalla de logeo/registro de la aplicación, teniendo que introducir obligatoriamente los datos solicitados.

  1. Estos dos botones te permiten cambiar entre logeo y registro.
  2. Estos campos deben rellenarse para poder finalizar el proceso.
  3. Botón para confirmar.

Historial de compra

Este boceto muestra la pantalla del historial de compra del usuario.

  1. Este botón te permite cerrar sesión.

Administración

Los siguientes bocetos muestran la gestión de la aplicación para un miembro del staff.

Menú principal de administración

Esta pantalla permite al staff navegar entre las distintas posibilidades que se les ofrece.

  1. Botón para alternar entre la vista de administrador y de usuario.
  2. Al pulsar en estos botones te permite la gestión de películas.
  3. Estos botones nos redirigen a gestión de cartelera.
  4. Se nos permite acceder a la gestión de salas.
  5. Este botón te permite cerrar sesión.

Gestión de películas

En este apartado encontramos los bocetos para añadir, modificar y eliminar una película de la base de datos.

Añadir/Modificar

  1. Datos obligatorios de la película
  2. Este botón sería para añadir o confirmar la modificación de la película, puesto que ambos usan el mismo formato para el formulario.

Selección de película para modificarla/borrarla

  1. Botón para modificar la película de la misma fila.
  2. Al pulsar aquí se puede borrar la película de la misma fila.

Gestión de cartelera

Aquí encontramos los bocetos para añadir, modificar y eliminar una sesión.

Añadir/Modificar

  1. Datos obligatorios de la sesión
  2. Botón que indica si al confirmar, será visible para el usuario o no.
  3. Este botón sería para añadir o confirmar la modificación de una sesión, puesto que ambos usan el mismo formato para el formulario.

Selección de una sesión para modificarla/borrarla

  1. Campos para una búsqueda con parámetros opcionales.
  2. Botón para modificar la sesión de la misma fila.
  3. Al pulsar aquí se puede borrar la sesión de la misma fila.

Gestión de salas

Aquí encontramos los bocetos para añadir, modificar y eliminar una sala.

Añadir/Modificar

  1. Datos de la sala iniciales.
  2. Este botón te genera una matriz con todas las butacas.
  3. Al pulsar en los asientos se alterna entre butaca y vacío.
  4. Este botón sería para generar o confirmar la modificación de una sala.

Selección de una sala para modificarla/borrarla

  1. Botón para modificar el tamaño y la forma de la sala.
  2. Al pulsar aquí se puede borrar la sala.
